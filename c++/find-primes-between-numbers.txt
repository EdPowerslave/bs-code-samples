// Aþaðýda da bir sayýnýn asal olup olmadýðýný kontrol eden fonksiyonu yazýyoruz. Bu iþlemi de sayýyý, 2 den baþlayýp o sayýnýn yarýsýna kadar olan her sayýya bölerek yapýyoruz. Çünkü matematiksel olarak bire bölünmesini saymýyoruz ve bir tam sayýya bölündüðünde yarýsýndan daha büyük bir sonuç çýkma ihtimali olmadýðýný biliyoruz. "isPrime" bu iþlemin sonucunu attýðýmýz evet/hayýr içeren deðiþken. 

int checkPrimeNumber(int n){
	bool isPrime = true;

        for(int j = 2; j <= n/2; ++j){
            	if (n%j == 0){ 
                 	isPrime = false;
                    	break;
             	}
   	}
        return isPrime;
}
